# 🐟 可可是杂鱼哦🥵♥ 编码器

一个基于传播杂鱼可可文化的文本编码/解码工具，使用特定的8个字符进行编码转换。

## ✨ 功能特点

+   **双向转换**：支持文本编码和解码
    
+   **特殊字符编码**：使用 `可` `是` `杂` `鱼` `哦` `🥵` `♥` `~` 8个字符进行编码
    
+   **实时转换**：输入时实时进行编码/解码
    
+   **一键操作**：支持复制和清除功能

## 🚀 快速开始

### 在线使用

### 本地部署

1. 在Releases中下载`run_keke.exe`
2. 双击运行`run_keke.exe`，在自动打开的网页中输入

## 📖 使用方法

### 编码文本

1.  在左侧"原始文本"框中输入要编码的文本
    
2.  右侧"编码文本"框会自动显示编码结果
    
3.  点击"复制文本"按钮复制编码结果
    

### 解码文本

1.  在右侧"编码文本"框中输入编码文本
    
2.  左侧"原始文本"框会自动显示解码结果
    
3.  点击"复制文本"按钮复制解码结果

### 自定义修改
1.  打开`keke.html`修改替换标题和替换如下数组内容，要求元素必须为8个

    ```
    const targetChars = ["可", "是", "杂", "鱼", "哦", "🥵", "♥", "~"];
    ```

2.  打开`run_keke.py`自定义`PORTS = [1443, 1433]`中的网页端口
    
3.  双击`pack.bat`打包文件

4. 双击`run_keke.exe`运行您的自定义修改版本

## 🔧 核心算法

使用基于8个特定字符的编码系统：

+   将文本转换为字节数据
    
+   使用大整数运算进行进制转换
    
+   支持所有Unicode字符的编码和解码
